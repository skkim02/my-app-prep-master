# 프로젝트 명: PREP Master (한경 사설 분석기)

## 1. 제품 개요 (Product Overview)
- 목표: 한국경제 신문의 사설을 자동으로 가져와 AI가 PREP 구조를 분석하고, 사용자가 논리적 글쓰기를 학습할 수 있는 환경 제공.
- 대상: 논술 준비생, 비즈니스 글쓰기 실력을 키우고 싶은 직장인.
- 핵심 가치: "읽기만 하는 사설에서, 내 논리로 만드는 사설로"

## 2. 주요 기능 (Key Features)

### ① 사설 데이터 자동 수집 (Scraper)
- 내용: 한국경제 '사설' 섹션의 최신 글 목록을 크롤링하여 날짜별로 그룹화하여 표시.
- 기술: Node.js 환경의 Cheerio 라이브러리 활용.
- 수집 항목: 제목, 본문, 작성일자, 원문 링크.
- 사용자가 원하는 사설을 선택하여 상세 분석 가능.

### ② AI PREP 구조 분석
- 내용: AI가 사설 본문을 자동으로 분석하여 PREP 구조를 추출.
- PREP 구조:
    - P (Point): 사설의 핵심 주장(결론)
    - R (Reason): 주장을 뒷받침하는 근거
    - E (Example): 제시된 사례나 데이터
    - P (Point): 최종적으로 강조하는 핵심
- 원문 하이라이트: 각 PREP 요소가 추출된 문장을 색상별 밑줄로 표시 (P-빨강, R-주황, E-초록, P-보라).
- 인터랙션: 분석 카드 hover 시 해당 원문 위치 강조.

### ③ Best Practice 예시
- 내용: AI가 사설 문체를 유지하며 PREP 구조로 재구성한 모범 답안 제공.
- 형식: 다른 사람에게 주장하듯 작성 ("~문제를 짚어보자", "왜 그런가", "실제로", "결론적으로").
- 목적: 사용자가 예시를 참고하여 본인만의 PREP 분석을 연습.

### ④ 분석 저장 및 관리
- 저장: 분석 결과를 LocalStorage에 저장.
- 목록: 헤더의 "저장됨" 버튼으로 저장된 분석 목록 확인.
- 삭제: 개별 분석 삭제 가능.
- 중복 방지: 동일 사설 중복 저장 불가.

## 3. 사용자 여정 (User Journey)
1. 접속: 사용자가 웹 사이트에 접속.
2. 목록 조회: [사설 목록 가져오기] 버튼 클릭 → 날짜별 사설 목록 로드.
3. 사설 선택: 원하는 사설 클릭 → 상세 분석 화면으로 이동.
4. 분석 확인: 좌측에서 원문(하이라이트 포함) 확인, 우측에서 AI 분석 결과 및 Best Practice 확인.
5. 저장: [분석 저장] 버튼으로 LocalStorage에 저장.
6. 복습: 헤더의 [저장됨] 버튼으로 이전 분석 기록 확인.

## 4. 기술 스택 (Technical Stack)
- Framework: Next.js 16 (App Router)
- Crawling: Cheerio (가볍고 빠른 HTML 파싱)
- UI/Design: Tailwind CSS 4 (빠른 스타일링)
- Language: TypeScript (안정적인 코드 관리)
- Storage: LocalStorage (클라이언트 저장)
- Deployment: Vercel

## 5. 버전 히스토리
- v1: 초기 버전 (단일 사설 분석, AI PREP 분석, 하이라이트)
- v2: 사설 목록 선택 기능 (날짜별 그룹화, 다중 사설 지원)
- v3: LocalStorage 저장 기능 (저장/삭제/목록 관리)

## 6. 향후 확장성 (Roadmap)
- AI 코칭: OpenAI API(GPT)를 연동하여 더 정교한 PREP 분석 및 피드백 제공.
- 사용자 입력: AI 분석과 별도로 사용자가 직접 PREP을 작성하고 비교하는 기능.
- 진도 관리: 날짜별 분석 기록을 대시보드로 시각화.
- PDF 내보내기: 완성된 분석 리포트를 PDF 파일로 저장 기능.
- DB 연동: 기기 간 동기화를 위한 서버 저장 기능.
