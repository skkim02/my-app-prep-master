# 프로젝트 명: PREP Master (한경 사설 분석기)

## 1. 제품 개요 (Product Overview)
- 목표: 한국 경제 신문의 사설을 자동으로 가져와 사용자가 논리적 구조(PREP)를 분석하고 직접 요약 연습을 할 수 있는 환경 제공.
- 대상: 논술 준비생, 비즈니스 글쓰기 실력을 키우고 싶은 직장인.
- 핵심 가치: "읽기만 하는 사설에서, 내 논리로 만드는 사설로"

## 2. 주요 기능 (Key Features)
### ① 사설 데이터 자동 수집 (Scraper)
- 내용: 한국 경제 '사설/칼럼' 섹션의 최신 글 1개를 크롤링.
- 기술: Node.js 환경의 Cheerio 또는 Puppeteer 라이브러리 활용.
- 항목: 제목, 본문, 작성일자, 원문 링크.

### ② PREP 구조 분석 가이드
- 내용: 사설 원문을 좌측에 배치하고, 우측에 PREP 입력 창 제공.
- 입력 필드:
    - P (Point): 사설의 핵심 주장(결론)은 무엇인가?
    - R (Reason): 주장을 뒷받침하는 근거는 무엇인가?
    - E (Example): 어떤 사례나 데이터가 제시되었는가?
    - P (Point): 최종적으로 강조하는 핵심은 무엇인가?

### ③ 실시간 저장 및 결과 확인
- 내용: 사용자가 입력한 내용을 로컬 스토리지 또는 DB에 저장하고, 완성된 PREP 문장을 하나의 리포트로 출력.

## 3. 사용자 여정 (User Journey)
1. 접속: 사용자가 웹 사이트에 접속.
2. 가져오기: [오늘의 사설 가져오기] 버튼 클릭 -> 최신 사설 로드.
3. 읽기: 좌측 화면에서 사설 원문을 정독.
4. 분석: 우측 PREP 필드에 각각의 내용을 요약하여 작성.
5. 완성: [제출/저장] 클릭 시 본인이 쓴 PREP 글쓰기 결과물 확인.

## 4. 기술 스택 (Technical Stack)
- Framework: Next.js (App Router)
- Crawling: Cheerio (가볍고 빠른 HTML 파싱)
- UI/Design: Tailwind CSS (빠른 스타일링)
- Language: TypeScript (안정적인 코드 관리)

## 5. 향후 확장성 (Roadmap)
- AI 코칭: OpenAI API(GPT)를 연동하여 사용자가 작성한 PREP 내용에 대한 피드백 제공.
- 진도 관리: 날짜별 분석 기록을 대시보드로 시각화.
- PDF 내보내기: 완성된 분석 리포트를 PDF 파일로 저장 기능.
